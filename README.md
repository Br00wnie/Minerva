# Minerva

## Desc

Web app for converting Markdown to PDF file with specified layout

## Stack

- **Frontend**: React + MobX, Axios
- **Backend**: Express, Sequelize
- **Database**: PostgreSQL
- **Other**: Vite, Slate

More technical information [here](dev.md)
